## 4. Krippendorff α + intersection-3way pool wiring

- [x] 4.1 `krippendorff_alpha(labels_per_coder, *, classes, missing_marker=None)` added to `src/trec_biogen/eval/metrics.py`. Uses Krippendorff (2011) standard formulation: per-unit numerator `(m_u² − Σ_c n_uc²) / (m_u − 1)` aggregated over `N = total non-missing labels`. No external dep.
